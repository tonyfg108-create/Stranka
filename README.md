# Jednoduchá vtipná HTML stránka

Tento projekt obsahuje jednoduchou, vizuálně koordinovanou HTML stránku s vtipným obrázkem a českým textem.

## Obsah

- `index.html` - hlavní HTML stránka
- `README.md` - tento soubor

## Popis

Stránka obsahuje:
- Nadpis s českým textem: "Podívejte se, jak je vše krásně jednoduché v dnešní době!"
- Vtipný obrázek (kočka v brýlích)
- Český text s vtipným komentářem
- Jednoduchý, čistý design s modrou barevnou schématem

## Jak použít

Jednoduše otevřete soubor `index.html` v jakémkoli webovém prohlížeči.

## Design

Stránka používá:
- Responsivní design
- Čisté, moderní rozložení
- Vizuálně koordinované barvy
- Dobře čitelné typografie
